python  打包
pip install pyinstaller
cd 到打包文件所在目录
pyinstaller -F xxx.py
dist/exe

打包多文件
pyinstaller [主文件] -p [其他文件1] -p [其他文件2] 
	--hidden-import [自建模块1] 
	--hidden-import [自建模块2]
For Example
pyinstaller [-F] main.py -p mysql.py -p other.py 
	--hidden-import mysql 
	--hidden-import other
其余依赖项只需在打包完成后，将它们放在打包后生成的.exe文件的同一个目录下即可
